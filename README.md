# CodeCraftHub

CodeCraftHub is a personalized learning platform built with Flask.

## Features

- Course Dashboard
- Add Course
- Edit Course
- Delete Course
- Search Courses
- Bootstrap 5 UI
- REST API

## Installation

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```

## API

GET /api/courses

POST /api/courses

PUT /api/courses/<id>

DELETE /api/courses/<id>
